### Capto - Administrate your bills on a new way




#### Setup Project

Clone project

```bash
  git clone https://github.com/raphaelaugustb/moneytimes/
```

cd Project directory

```bash
  cd MoneyTimes
```

## Seutup back-end

Install depedencies

```bash
  cd backend/moneytimes && mvn clean install package 
```
Run database on container

```bash
  docker compose up
```
Run api

```bash
  mvn spring-boot:run 
```
